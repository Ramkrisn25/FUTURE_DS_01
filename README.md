# FUTURE_DS_01 – Business Sales Dashboard

## 📊 Overview
This project analyzes e-commerce sales data to:
- Identify best-selling product categories
- Highlight monthly sales trends
- Show top-performing products
- Summarize KPIs like Total Sales and Profit

Built as part of my internship at **Future Interns**.

---

## 🛠️ Tools Used
- Power BI Desktop

- Total Sales by Category
- Monthly Sales Trend
- Top 10 Products by Sales
- Profit by Region
- KPI Cards for Sales, Profit, Quantity
